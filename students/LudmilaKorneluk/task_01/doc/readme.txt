Описание проекта
Веб-приложение для рассылки писем с информацией о рекламных кампаниях с семантической разметкой, адаптивностью и доступностью.

Страницы приложения:
Главная страница (index.html) - Лендинг со списком всех кампаний и их статусом
Карточка кампанрии (campaign.html) - Карточка кампании с возможностью предпросмотра отправляемых писем
Страница аналитики (metadata.html) - Детальная информация об измении карточки кампании с заметками


Структура проекта
WebApplication1/
├── index.html          # Главная страница
├── campaigns.html      # Страница кампаний
├── analytics.html      # Страница аналитики
├── styles.css          # Стили
└── README.md
