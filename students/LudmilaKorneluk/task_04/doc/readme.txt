Лабораторная работа 04 - Центр рассылок «Не спам, а ценность»
Описание проекта
ASP.NET Core Razor Pages приложение для управления email-рассылками с аутентификацией через сессии.

Функциональность
Главная страница с преимуществами сервиса

Страница со списком кампаний рассылок

Детальная страница кампании

Система аутентификации (фиктивная)

Страница аналитики с защищенным доступом

Привязка моделей для данных кампаний

Запуск проекта
bash
dotnet run --project Task04.Web --urls="http://localhost:5000"
Приложение будет доступно по адресу: http://localhost:5000

Тестовые данные для входа:

Логин: admin

Пароль: password
