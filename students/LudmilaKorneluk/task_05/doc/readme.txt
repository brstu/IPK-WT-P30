Лабораторная работа 05. DI/слои, Identity и файлы (новичок)


## Зависимости между слоями

- **Web** → Application, Infrastructure
- **Application** → Domain
- **Infrastructure** → Application, Domain
- **Domain** - независимый слой

## Роли и сценарии

### Роли:
1. **Admin** - полный доступ ко всем функциям
2. **User** - базовый доступ с ограничениями

### Сценарии:

#### Регистрация пользователя
1. Пользователь переходит на страницу регистрации
2. Заполняет email и пароль
3. Система создает учетную запись с ролью "User"
4. Пользователь перенаправляется на главную страницу

#### Вход в систему
1. Пользователь вводит email и пароль
2. Система проверяет учетные данные
3. При успешной аутентификации создается сессия
4. Пользователь перенаправляется на dashboard

#### Загрузка файла
1. Авторизованный пользователь переходит в раздел файлов
2. Выбирает файл для загрузки (макс. 2MB)
3. Система проверяет тип и размер файла
4. Файл сохраняется с безопасным именем
5. Запись о файле добавляется в базу данных

#### Скачивание файла
1. Пользователь просматривает список своих файлов
2. Нажимает кнопку скачивания
3. Система возвращает файл с правильными заголовками

#### Доступ к административной панели
1. Только пользователи с ролью "Admin" могут получить доступ
2. Система проверяет роль пользователя
3. При отсутствии прав доступа возвращается ошибка 403
