Отчет по лабораторной работе №2
Основы JS/TS и регулярные выражения
Цель работы
Разработка формы регистрации с валидацией полей средствами TypeScript и регулярных выражений.

Выполненные задачи
1. Валидация полей формы
Email: проверка формата с регулярным выражением /^[^\s@]+@[^\s@]+\.[^\s@]+$/

Телефон: валидация белорусского формата (+375 XX XXX-XX-XX) и проверка кодов операторов (29, 33, 44, 25)

Пароль: проверка сложности (минимум 8 символов, заглавные/строчные буквы, цифры)

Навыки: проверка выбора хотя бы одного варианта

2. Реализованные функции
Валидация в реальном времени (на событие blur)

Форматирование телефонного номера при вводе

Отображение ошибок под соответствующими полями

Визуальная индикация невалидных полей

3. Технологии
TypeScript для типизации и организации кода

Регулярные выражения для валидации email и телефона

Vite для сборки проекта

CSS Grid/Flexbox для адаптивной верстки

Запуск проекта
bash
npm install
npm run dev
Результат
Форма регистрации с комплексной валидацией всех полей, работающей в реальном времени с immediate feedback для пользователя.
